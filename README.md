## 📖 О проекте
  
`email_validator` — Проект представляет собой Python-пакет для проверки валидности email-адресов. Он разделен на логику валидации и интерфейс командной строки для удобства использования и тестирования.

## 📂 Структура проекта

```
email_validator_project/
├── email_validator/     # Папка пакета
│   ├── __init__.py      # Инициализация пакета
│   └── validator.py     # Логика валидации и точка входа (main)
│
├── tests/               # Папка для тестов
│   ├── __init__.py
│   └── test_validator.py # Тесты для логики валидации
│
└── README.md            # Файл документации
```

## ⚙️ Примеры работы

- Проверка валидного адреса:
Введите email-адрес для валидации: user@gmail.com

✅ Email-адрес 'user@gmail.com' является валидным.

- Проверка некорректного адреса:
Введите email-адрес для валидации: user@@mail.ru

❌ Неверный формат email: user@@mail.ru

- Работа с несуществующим доменом:
Введите email-адрес для валидации: user@nonexistant-domain.xyz

❌ Домен для email 'user@nonexistant-domain.xyz' не найден или недоступен.

## 💡 Использование

**Шаг 1** Запустите программу в терминале.

```bash
python -m email_validator.validator
```

**Шаг 2** Введите email для проверки.
```
Введите email-адрес для валидации: user@mail.ru
```
**Шаг 3** Оцените результат.
```
✅ Email-адрес 'user@mail.ru' является валидным.
```
Возможные ошибки:

❌ Неверный формат email: example
❌ Домен для email 'user@fake-domain.ru' не найден или недоступен

## 🧠 Логика валидации

**Алгоритм состоит из двух этапов:**
```
- ⏱ Проверка формата (регулярное выражение)
- DNS Проверка существования домена (опционально)
```
**Особенности реализации:**
```
Используется библиотека socket для DNS-запросов.
Время ожидания ответа ограничено (таймаут 5 секунд).
```
**Исключения оформлены иерархически:** EmailValidationError → InvalidEmailFormatError, DomainNotFoundError.

## 🛠️ Технологии

- **Python 3.9+**
- Стандартная библиотека: re (регулярные выражения), socket (сетевые запросы)
- Архитектура: объектно-ориентированное проектирование (class EmailValidator)
- Управление потоком: argparse (интерфейс командной строки)
